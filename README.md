E-commerce Revenue \& Profitability Analysis



This project analyzes an e-commerce dataset to identify profit leaks, inefficient categories, and the impact of discounts on profitability.



Tools Used

\- Power BI

\- SQL

\- Python



What I Did

\- Analyzed revenue, profit, and margin by category and segment

\- Examined yearly trends

\- Studied how discount levels affect profitability

\- Built a simple automated monitoring workflow using Python
